Detailed Description

CellDouble is a class which extends the abstract class Cell Every instance of this class takes care of 1 double value
This class allows:

  • change current double value - setValue
  • get current value as a double - getValue
  • get string representing current value - getDisplayableString and getConstructString
  • clone this object - clone

std::string CellDouble::getDisplayableString ( )
virtual
Returns
Casts current value into a string. String is formatted, so that it can be displayed in best possible way. This means:
  • 1) without extra (not meaningful) zeroes in the beginning of the whole part of the number or at the end of its floating part
  • 2) does not display the floating part if it's equal to 0
