Detailed Description

CellString is a class which extends the abstract class Cell Every instance of this class takes care of 1 string value (object of type std::string)
This class allows:

  • change current int value - setValue
  • get current value as a int - getValue
  • get pure string value of current string - getDisplayableString
  • clone this object - clone
