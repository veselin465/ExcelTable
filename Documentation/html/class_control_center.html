Detailed Description

ControlCenter is a class which aims to centralize all the main logic of this app. An instance of this class is required in order to work with the functionality of class Table in the most user friendly way
This class can execute a command based on a provided string - executeCommand

◆ executeCommand()

const std::string ControlCenter::executeCommand ( const std::string &  commandLine)

Given a command, this function tries to execute it. 3 Possible outcomes:

  • command execution was successful, but had no purpose of giving feedback, so it returns empty string
  • command execution was successful and it gives a feedback or displayable information related with the executed command
  • command execution was not successful, so it throws an exception any or does not catch any thrown exception
Exceptions
invalid_argumentThrown to signal that the request failed
