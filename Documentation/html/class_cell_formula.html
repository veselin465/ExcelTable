Detailed Description

CellFormula is a class which extends the abstract class Cell Every instance of this class takes care of one formula. A formula represents an algebraic expressing, whole return value is a floating number.
A formula consists of operands (real numbers), unary operators (+,-) and binary ones (+,-,*,/,^)
This class allows:

  • set and change current formula - setValue
  • calculate the value this formula has - automatically calculated when setValue is called or calculated using the last remember formula when called recalculate
  • get current value as a double - getValue
  • get string representing current value - getDisplayableString and getConstructString
  • clone this object - clone

◆ calculateFormulaRecursively()

double CellFormula::calculateFormulaRecursively ( const std::string &  value)
private

Given a string with an algebraic expression, this function analyzes it, determines operation priorities and smartly divide the problem into 2 smaller problems which are again sent to this function to calculate them until an operand is reached. The following algorithm is applied

  • 1) Start searching operations, where it first searches low priority, then high priority ones
  • 1.1) When found, the entire string is split into 2 halves, without the operation itself
  • 2) Then calls the algorithm on the 2 separated parts, gets the value from them and
  • performs the appropriate operation on the parts
    Note
    Uses mathematically proven and tested methods to get the correct answer NOTE: This recursion depends on certain string rules to work properly, like:
  • 1) provided string is entirely whitespace trimmed - trimEntirely
  • 2) if provided string contains references in table, they need to be calculated - extractCellValue
  • 3) used operands and operations are appropriately used (otherwise throws exception)
Exceptions
invalid_argument- thrown if the expression is invalid. Can be thrown if:
  • brackets without any expression in it is contained in the expression
  • An operation does not have the required number of operands

expressionto calculate
Returns
wanted result casted to a floating number

double CellFormula::extractCellValue ( size_t  row,
size_t  column 
)
private

Given a position in 2D coordinates (row and column), returns the value on that position according to the provided table. The following calculate dependencies apply:

  • 1) empty cell (or one outside of the provided table) is considered 0
  • 2) string cells are considered as 0, even if they have number value
  • 3) formula cells with errors share the error in this formula as well
  • 4) int, double and formula without error gives right away the value they hold
