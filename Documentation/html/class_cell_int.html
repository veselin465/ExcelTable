Detailed Description

CellInt is a class which extends the abstract class Cell Every instance of this class takes care of 1 integer value
This class allows:

  • change current int value - setValue
  • get current value as a int - getValue
  • get string representing current value - getDisplayableString and getConstructString
  • clone this object - clone
